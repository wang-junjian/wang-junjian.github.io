## iPhone
### Safari长截图
1. 在Safari浏览器打开任意网页，同时按下音量调高键+锁屏键进行截屏。
2. 点击左下角的缩略图，在预览界面上选择【整页】选项。
3. 点击左上角的【完成】，然后保存PDF文件就可以了。

* [iPhone如何截长图？](https://www.163.com/dy/article/H5B36C940552C3N4.html)
