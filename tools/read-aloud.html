🎙️ Read Aloud 使用帮助

Read Aloud 是一款基于浏览器 Web Speech API 的本地文本朗读器。无需上传文件，所有处理都在浏览器内完成，支持中英文长文朗读、自动分句、当前句高亮与进度同步。

快速开始

  • 将长文粘贴到文本区，点击 ▶ 朗读 即可开始。
  • 朗读过程中文本区自动切换为阅读视图，当前句子会高亮显示并自动滚动到视野中央。
  • 点击 ⏹ 停止 可结束朗读并恢复文本编辑。

点击跳转播放

  • 在阅读视图中，单击任意句子即可立即从该句开始播放，方便回退、重听或跳到指定位置。
  • 在编辑视图中，先把光标放到目标段落，再点击 ▶ 朗读，朗读会从光标所在段落开始。

语音与语言

  • Safari 默认优先选择 Samantha (en-US)，Chrome 默认优先选择 Ava (en-US)。
  • 中文内容会优先使用系统中文语音；也可在“语音”下拉框中手动选择。
  • “语言”设为自动时，系统会根据文本中汉字与英文字母的比例自动判断。

段落与空白行

  • 长文会按段落和句子自动切分，避免单次文本过长导致浏览器截断。
  • 朗读时会自动去除空白行，让阅读视图更紧凑；停止后会恢复原始文本与段落布局。

快捷键

  • 在编辑区内按 Ctrl / ⌘ + Space 播放 / 暂停。
  • 焦点在编辑区外时，按 Space 也可快速切换播放 / 暂停。
  • 按 Esc 可关闭此帮助窗口。

提示

  • 若“语音”列表为空，请稍等片刻再试，部分浏览器需要异步加载语音包。
  • 遇到朗读异常时，点击 ⏹ 停止 后重新播放即可恢复。
