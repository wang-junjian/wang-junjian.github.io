## 选择 Python 解释器
1. 通过 Shift + Command + P 快捷键，打开命令面板。
2. 输入 ```Python: Select Interpreter``` ，回车。
3. 选择您想使用的环境。

**在状态栏上可以单击进行切换不同的环境**
